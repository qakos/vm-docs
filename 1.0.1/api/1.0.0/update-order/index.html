Update Order

POST 
/connect/update-order

Updates an existing order with new status or shipment details. Editable fields include order status, shipment details, and item quantities.

Request​

Query Parameters

    order_number stringrequired

    The unique order number to update

  • application/json

Body

required
    order objectrequired
    orderHeader object
    status string

    Possible values: [accepted, canceled, shipped, archived]

    New order status

    actualShipments object[]

    Array of shipment details (required when status is 'shipped')

  • Array [
    • carrier string

    Shipping carrier

    shipmentNumber string

    Shipment number

    date date-timerequired

    Shipment date

    trackingNumbers string[]required

    Tracking numbers

    items object[]
  • Array [
    • sku stringrequired

    Product SKU

    qty integerrequired

    Quantity

  • ]
  • ]
